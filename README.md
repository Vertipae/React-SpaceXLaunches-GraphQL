# React-SpacexLaunches-GraphQL
Cool SpaceX launch app

## Link: https://glacial-stream-48128.herokuapp.com/

```
# Install dependencies
npm install
cd client && npm install

# Run server & client (:3000 & :5000)
npm run dev

# Server only (:5000)
npm run server

# Client only (:3000)
npm run client

# Graphiql - http://localhost:5000/graphql
```
